# Tesina — Maria Sveva Fuzio

Raccolta digitale della tesina per l'esame conclusivo della Scuola secondaria di 1° grado di **Maria Sveva Fuzio**, presentata come app web interattiva costruita con [Streamlit](https://streamlit.io).

## Tema

*Le mille e una notte* come filo conduttore interdisciplinare: dal confronto letterario con Italo Calvino alla traduzione di Antoine Galland, dalla geografia e spiritualità del mondo islamico alla geopolitica del Medio Oriente, fino all'arte (Magritte), la musica, la danza e i diritti delle donne (Malala).
